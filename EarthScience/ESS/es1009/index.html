地球科学导论–开始

# 第 0 章 地球科学导论

# 什么是地球科学

它是试图了解地球及其周围空间的所有学科的总称，包括：

  • 地质学
    • 自然地质学 物质组成，了解地表附近的过程
    • 历史地质学 了解地球起源与演化
  • 海洋学 涉及所有领域，综合化学、物理学、地质学、生物学
  • 气象学 研究大气圈及其作用过程
  • 天文学 地球在宇宙空间中的位置

# 圈层

4 个圈层间相互作用。

# 水圈

海洋：71% 地球表面，地球上 96% 的水
地下淡水，河流，湖泊，冰川。
大气、云层中的水和水蒸气。

# 大气圈

防御太阳辐射，能量交换导致天气与气候。

# 生物圈

地表附近，天空，海底。维持自然环境。

# 岩石圈

地表延伸到地心，厚约 6400 千米。地球内部包括

  • 地核 高密度
  • 地幔 密度较小的中层
  • 地壳 密度很小，很薄

岩石圈：刚性外层 包含地壳和地幔最上层

软流圈的岩石是软塑性的，根据热量分布缓慢移动。

# 地球系统

四个圈层相互作用，形成的复杂整体称为地球系统。采用跨学科方法了解与解决全球性环境问题。

时间和空间尺度：毫米至数千千米，毫秒到几十亿年。
地球系统的能量：太阳驱动的外力作用，地球内部。
人类和地球系统：人是地球系统的一部分。

# 时间和空间尺度

常常面对巨大的时间跨度，地质学家：1 亿年前为 “近”，1000 万年演示样品为 “年轻”。

# 资源和环境问题

资源包括水、土壤、矿产、能源等。地球科学探讨资源的形成和发展，及维持资源平衡、开发利用对环境的影响。
无生命的统称为自然环境，包括水、空气、土壤和岩石，以及温度、湿度和阳光等。现在环境通常指人和自然环境间的关系。
环境问题有局部性的，区域性的，以及全球性的。除人为加剧的，还有自然灾害。

# 科学探索的本质

科学是形成知识的过程，所有科学基于自然界是以统一和可预测的方式运行的这一假设，科学的最终目标是发现自然界中的潜在规律，并利用这些知识作出可以期望的预测。

观测数据获得事实，根据现象提出试探性的说法或假说，假说经过审查并去除有争议的内容上升为科学理论。这些通常称为科学方法，是涉及创造力和洞察力的一种尝试。

最具说服力的科学理论也只是自然界的简化解释。
