地球科学导论–动荡的海洋

# 第 10 章 动荡的海洋

# 海洋的表层环流

表层环流由风力形成，能量通过摩擦力由风传给海水。海洋表层环流模式不仅与全球的风况相符，同时也受主要陆地分布和地球自转的影响🌍。北半球右偏，南半球左偏。

# 洋流模式

大规模环形运动的洋流在海洋表面占主要地位。大洋盆地内巨大水流的回旋称为洋流。
五大环流：北太平洋环流，南太平洋环流，北大西洋环流，南大西洋环流，印度洋环流。
每个环流中心位于南北纬约 30° 的亚热带区域，故也称亚热带环流。

  1. 北太平洋洋流
    包括北赤道洋流、黑潮洋流、北太平洋洋流和加利福尼亚洋流。
  2. 北大西洋洋流
    北赤道洋流穿过加勒比海向北偏离成为墨西哥湾流，向东偏离北卡罗莱那州海岸到达北大西洋，渐渐变宽变慢成为北大西洋暖流。
    北大西洋洋流接近西欧时分为两部分，一部分向北途径英国、挪威和冰岛，另一部分向南成为加那利洋流。
    洋流的环形运动使得中间区域不存在洋流，在北大西洋，这部分平静的水域称为马尾藻海。
  3. 南半球洋流
    大洋盆地的流动模式与北半球相似。
    西风流是完整环绕地球的唯一洋流。
  4. 印度洋洋流
    主要在南半球，表层洋流模式与南半球洋盆中的环流模式类似。然而北半球的小部分印度洋会受季风的影响。

# 洋流影响气候

地球获得的太阳能与其表面向空间辐射的热量相等，低纬度地区获得大于失去，高纬相反，地区间存在大规模的能量交换。风和洋流的能量转移补偿了不同纬度地区的能量不平衡，其中海水运动的能量约占 1/4，风占 3/4。

  1. 暖流
    向两级运动的暖流的调节作用是众所周知的。北大西洋暖流使得英国和大部分西欧地区的冬天要比它们所在纬度地区的气温高。
  2. 寒流
    寒流对热带地区或中纬度地区的夏季影响最大。
    寒流近海区域使得低空大气变冷，使大气稳定，不会上升形成云，进而降水减少，加重干旱。如洋流对大陆西海岸的热带沙漠有巨大影响。
    寒流的出现会使温度接近或达到露点（水蒸气凝结的温度），这些地区会有相对高的湿度，造成多雾现象。因此寒流存在的热带沙漠也可以是相对寒冷和潮湿的地区。

# 上升流和深海环流

有些垂直运动与风生表面流有关，而深海环流则受密度差的强烈影响。

# 海岸上升流

上升流指深部的冷水上升并替换温暖的表层海水，是一种常见的风生垂直运动。海岸上升流发生在风吹向赤道并平行与海岸的地区。沿海的风与科里奥利效应一起使得表层海水从海岸移走，被来自表层之下的 “上升” 海水代替，使得海岸附近表层海水的温度更低。
上升流会将更大溶解度的营养物质带至海洋表层，促进微型浮游植物的生长，进而增加鱼类和其他海洋生物的数量。

# 深海环流

深海环流在垂向上有明显分层，故能使深水物质充分混合。垂向分层是由水体密度差异导致的，密度差是由海水温度和盐度不同引起的，因此也称热盐环流。
高纬度地区表层海水首先卷入深海环流。表层海水冷形成海冰，剩余海水盐度增加直到下沉，在海底以缓慢的水流形式在整个大洋盆地运动。

# 海岸：一个动态界面

界面是指系统不同部分相互作用的边界。大陆和大洋沿海岸汇聚的过程会产生快速变化的地貌。
海岸受波浪侵蚀影响，受多种地质作用。
