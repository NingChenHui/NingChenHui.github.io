Note#04 Foundations of Earth Science

# 第 10 章 动荡的海洋 续

# 波浪

波浪是沿海洋与大气间的界面传播的能量，通常会被风暴传播几千千米的距离。风生浪提供了改变海岸形状的大部分能量，海岸是不可抗拒力量遭遇不可移动物体的地方，两者冲突有时会相当剧烈。

# 波浪的特征

大多数波浪从风中获得能量并运动。风速小于 3km/h 只会出现很小的波浪，风速加大时会逐渐形成更加稳定的波浪并前进。波浪的要素和普通的正弦波动差不多。波峰和波谷之间的垂直距离称为波高。

波浪的波高、波长和周期取决于三个因素：

  1. 风速
  2. 风力作用的时间
  3. 风浪区，即风穿过开阔水域的距离。

由于风给水传递能量，波浪的高度和倾斜度增加，波浪最后会到达高到突然倾覆形成碎波（称为白浪）的临界点。对于某个特定风速而言，它存在一个最大的风浪区和持续时间，超出这一时间后波浪的大小将不会增加。当某个风速到达最大的风浪区和持续时间时，这时的波浪是 “完全成熟的波浪”，此时波浪接收风的能量和白浪颠覆损失的能量相同。

当风停止或改变方向，或波浪离开风暴区时，波浪会不依赖风继续运动，还会变成涌浪，高度更低长度更长，能把能量携带到遥远的海岸。由于同时存在独立的波浪，海面会形成复杂不规则的运动模式。

# 圆周运动

波浪行进时，水体是通过圆周运动传递能量的。波形传递时，各个水滴基本上在做圆周运动，这和风吹麦浪很像。风传给海水的能量不仅沿海面传授，而且向下传送。1/2 波长下方水的运动可以忽略不计，这一深度称为浪基面。

# 碎波带的波浪

当水深等于浪基面时，波浪开始 “触底”。海底会干扰水在波浪底部的运动，并减缓其前行的速度。随着波浪继续向海岸推进，运动更快且离海岸更远的波浪将追赶前面的波浪，导致波长减小。随波速波长减小，波浪高度稳定增加，最后向前倒塌或破碎，继续向岸边推进。
破碎波浪形成的湍流称为碎波。大陆边缘的碎波带中，坍塌碎波形成的湍流称为冲流，冲向摊坡。冲流能量耗尽时，从海滩流回碎波带的海水称为回流。

# 海滩和海岸的形成过程

海滩是海洋或湖泊向陆边缘的泥沙淤积。海滩可沿笔直的海岸延伸几十或几百千米。在不规则的海岸带，海滩可能仅出现在水域相对平静的海湾。

海滩由当地丰富的物质组成。有些海滩的沉积物源自临近的陡崖或附近海岸山脉的侵蚀，有些海滩的则是由河流向海岸搬运而来。尽管很多海滩的主要成分是耐侵蚀的石英颗粒，但有些也可以是其他物质，如贝壳碎片、玄武质熔岩颗粒。

波浪会不断冲刷并移动海滩。

# 波浪侵蚀

无风天气波浪作用很小，但风暴天气下高能量的巨浪拍击海岸时非常恐怖。受巨大冲击的地方很快会出现裂缝。
磨蚀作用也很重要，即海水对岩石的切割和磨削。

# 海滩上的泥沙运动

  1. 垂直于海岸的运动
    波浪活动较小时，海滩上大多是冲流，导致沙粒向滩面的净移动。高能量波浪袭来时，强烈的回流侵蚀沙滩，导致沙粒向大洋的净移动。

  2. 波浪折射
    波浪的折弯称为波浪折射，它会影响能量沿海岸的分布，强烈影响侵蚀的位置和侵蚀程度，以及沉积物的运移和沉积。
    大多数波浪以较小的角度移向海岸，当最靠近海岸的波浪因为触底而变慢，而深水仍然全速前进，最后以近乎平行海岸的方式趋近海岸。
    波浪折射导致波浪的能量集中于海岬的两侧和末端，使得海湾处的冲击力变弱。长期的海岬侵蚀和海湾沉积会使得不规则的海岸变直。

  3. 沿岸搬运
    尽管波浪会发生折射，但大多数波浪仍以较小的角度到达海岸，因此每个碎波的上冲海水都会以某个倾角到达海岸，而回流则直接沿滩坡向下运动。波浪会呈锯齿状沿沙滩表面输送沉积物，称为沿滩漂移。
    以小角度靠近海岸的波浪也可在平行于海岸流动的碎波带产生水流，因此与沿滩漂移相比会运移更多的沉积物。

# 海岸的特点

# 侵蚀特征

  1. 海蚀崖
    波浪冲击沿海基岩经切削作用形成的
  2. 海蚀台地
    由后退的悬崖演变而成，相对平坦。随波浪冲击加宽。
  3. 海蚀阶地
    构造作用将台地抬升到海平面上形成阶地，是沿海开发的理想地点。
  4. 海蚀拱
    海岬被波浪冲击，较软的破碎岩石最初形成海蚀洞，两侧反向海蚀洞贯穿后形成海蚀拱。
  5. 海蚀柱
    海蚀拱顶部坍塌后的残余岩体，最终会被侵蚀殆尽。

# 沉积特征

包括沙嘴、沙洲、连岛沙洲、堰洲岛等。

# 不断变化的海岸

不规则 -> 加大不规则程度 -> 构造稳定 -> 规则海岸

# 潮汐

潮汐指海面的日变化，是由月球和太阳对地球的引力造成的。

# 成因

月球的引力会导致地球的近月侧形成潮隆，而远月侧也会形成同样大小的潮隆。近侧引力大于远侧印力，引力差导致地球稍微拉长。
一天内月球变化很小，潮隆位置不会随地球自转改变。潮隆每 29.5 天会随月球的旋转移动一个循环。
一天内两次高潮一般不相等。北半球的第一次高潮要大于半天后的高潮，南半球相反。

# 月潮周期

太阳对潮汐的影响约为月球的 46%。
地球、月亮、太阳在一条直线上时，引力叠加形成大潮，每月出现两次。在直角边上时，引力呈正交关系抵消，称为小潮，每月出现两次。每次潮汐间隔一周。

# 潮汐类型

前面是根本原因和类型，各地海岸的形状、大洋盆地的结构和水深也都会对潮汐产生很大影响。
分别有全日潮、半日潮、混合潮三种。

# 潮汐流

指伴随潮涨潮落的海水的水平流动。
涨潮流和退潮流间很少或不出现潮汐流的时期称为平潮。受这些交替潮流影响的区域称为潮滩。
潮汐流在开阔的大洋中并不重要，但在海湾、河口、海峡和其他狭窄区域的流速很快。
