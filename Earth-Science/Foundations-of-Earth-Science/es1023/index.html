Note#02 Foundations of Earth Science

# 第 9 章 海洋：最后的净土

# 浩瀚的海洋世界

地球表面积 5.1 亿平方千米，其中 71% 是海洋和边缘海，北半球陆地偏多。

四个主要的洋盆：

  1. 太平洋，最大、最深。
  2. 大西洋，约为太平洋一半。
  3. 印度洋，比大西洋稍小，主要是南半球水体。
  4. 北冰洋，约为太平洋 7%。

# 海水的构成

海水中的溶解物质包含氯化钠、其他盐类、金属、溶解性气体等。

盐度指溶解在水中的固体物质的总量。平均盐度为 3.5%。初氯化钠外海水中较丰富的盐类为硫酸钠SO42−SO_4^{2-}SO42−​、氯化镁Mg2+Mg^{2+}Mg2+、氯化钙Ca2+Ca^{2+}Ca2+、氯化钾K+K^+K+。

海盐的来源主要是陆地岩石的化学风化。第二来源是地球内部，火山喷发会释放出大量的水和溶解气体。海水的总体构成在历史时期保持相对恒定。

海水是均匀混合的，主要成分的相对丰度基本上是恒定的，因此盐度的变化主要是由溶液中水含量的变化引起的。
如降水、地表径流、冰川融化、海冰融化，会将淡水添加到海洋中，使盐度降低。如蒸发作用和海冰形成会使盐度增加。蒸发速率高的高盐度地区位于干燥的亚热带地区，而在中间纬度和赤道附近的区域，高降水量会稀释海水。两极地表盐度的变化是由海冰的形成与融化引起的，冬天盐度大。

# 温度、密度随深度的变化

# 温度

太阳使表面海洋变得温暖，表面海水的温度在热带地区更高，向两级会变得更低。
低纬度的 300-1000 米水深处温度急剧变化（从 20 + 跌到 5），因此称为温跃层，它为许多类型的海洋生物创造了垂直屏障。
1000 米之后，温度只比结冰温度高几度。
高纬度地区温度不快速变化，纵向上等温。有些高纬水体在夏天时会有小幅升温，经历极弱的季节性温跃层。中纬度地区会有更明显的季节性等温层。

# 密度

密度差会引起大面积的海水沉浮。影响密度的因素主要是盐度和温度。盐度提高密度增加，温度增加使海水扩张，密度降低。

温度对表层海水密度影响最大，仅在海洋极端极地地区温度很低相对恒定，盐度显著影响密度。

低纬度地区密度在海水表层从低密度开始，随温度增加快速升高，在 1000 米深度处达最大值。300-1000 米为密度跃层，有很高的重力稳定性，是防止上部低密度海水与下部高密度海水混合的重要屏障。

高纬度地区不存在密度跃层，纵向上等密度。

# 海洋分层

除了具有高蒸发率的浅内陆海外，在海洋最深处也可发现密度最大的海水。海水可划分为三个主要的层：

  1. 表面混合层，海水表面接收太阳能，温暖，仅占 2%。混合的海水通过波浪、紊流和潮汐散步表层热量，具有几乎相同的温度。该层厚度和温度会随纬度和季节变化。
  2. 过渡层，包括温跃层和密度跃层，占 18%。
  3. 深部层，高密度的冷海水，占 80%。阳光无法到达这里，密度高数值恒定。

高纬度地区这三层结构不存在，表层形成较冷的高密度海水，下沉会导致深海洋流。

# 海底图像

三个主要地貌单元：大陆边缘，深海盆地和洋中脊。

# 大陆边缘

陆地的外部边缘，陆壳过渡到洋壳的位置。

# 被动大陆边缘

指远离板块边界的不活跃区域，与强烈地震和火山活动无关。

  1. 大陆架
    海岸线向深海盆地延伸的倾斜潜面，主要由被沉积岩覆盖的陆壳和临近地块剥蚀的沉积物组成。具有丰富的石油、天然气，是重要渔场，有重要经济意义和政治意义。所以咱们的南海属于大陆架吗？
  2. 大陆坡
    区分陆壳与洋壳边界的较陡构造，平均倾角 5°，有些超过 25°。
  3. 大陆隆起
    并入逐渐倾斜区域的大陆坡，由厚厚的沉积物组成。
  4. 海底峡谷和浊流
    切割大陆坡的深陡峡谷，可能一直被浊流侵蚀。

# 活动大陆边缘

位于会聚板块边界，大洋岩石圈俯冲到大陆前缘下方位置，形成深海沟。

# 深海盆地的特征

深海盆地位于大陆边缘与洋脊之间，约占地球表面积的 30%。

# 深海沟

长且窄的海槽，是海洋中最深的区域，大多数沿太平洋边缘分布，深度可达 1000 米以上。
海沟是板块会聚的位置，大洋岩石圈在此处向下俯冲到地幔。与海沟平行排列的一系列弧状火山被称为火山岛弧。

# 深海平原

海底深且平的地貌单元。所有海洋中都存在。

# 海底火山结构

  1. 海底的火山称为海山，海山要比周围的地形高数百米。
  2. 如果在板块运动岩浆源消失前，火山已生长得足够大，就会形成火山岛。
  3. 不活动的火山岛由于风化和侵蚀降低到与海平面相近的高度，随运动板块携带至水面下，称为海底平顶山。
  4. 海洋高原类似于陆地上的熔岩高原，由上升地幔柱熔融产生的大量玄武质岩浆形成。

# 洋脊

沿发育良好的汇聚板块边界，海底被抬升并生成较宽的线状隆起称为洋脊、隆起或洋中脊。
洋脊所在位置广泛存在断裂作用、地震、高热流和火山活动。
洋脊是由地幔产生的新洋壳的岩浆上涌形成的。
洋脊系统某些段的轴部会出现大裂谷。

# 海底沉积物

  1. 陆源沉积物 风化并运移到海洋的大陆岩石颗粒。
  2. 生物源沉积物 海洋动物和藻类植物的外壳与骨骼。
  3. 水成沉积物 海水化学反应结晶而成的矿物。
